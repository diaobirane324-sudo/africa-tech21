PROTOTYPAGE

INTRODUCTION

Le projet BINV (Bâton Intelligent pour les Non-Voyants) version améliorée V360 a pour but de faciliter la vie des aveugles. Il s'agit d'un bâton qui alerte ces derniers contre les obstacles : trous, murs, roches, etc.

Après avoir identifié l'obstacle, une voix est prononcée dans différentes langues (française, wolof, anglaise, etc.) à l'aide d'un écouteur pour mieux interpréter le message.

NB : Pour ce modèle, l'écouteur est avec fil. Les appareils avec Bluetooth ne sont pas compatibles pour cette version, mais cela pourra être envisagé dans les versions futures.

FONCTIONNEMENT DU SYSTÈME

Le fonctionnement du BINV repose sur une chaîne de traitement simple mais efficace :

  1. Capture d'images : La caméra intégrée à l'ESP32-CAM capture en temps réel l'environnement devant l'utilisateur.
  2. Analyse IA : Un algorithme d'intelligence artificielle analyse les images pour détecter les obstacles (murs, trous, roches, etc.).
  3. Décision : Si un obstacle est identifié, le système prend une décision : alerter l'utilisateur.
  4. Alerte vocale : Le DFPlayer Mini lit un fichier audio MP3 stocké sur la carte microSD, qui est diffusé via le haut-parleur ou l'écouteur.
  5. Message personnalisé : L'alerte est adaptée à l'obstacle détecté (ex: "Attention, trou devant", "Obstacle à droite", etc.).

COMPOSANTS ÉLECTRONIQUES

Pour l'équipe technique électro : Voici le rôle de chaque composant au sein du projet BINV.

ESP32-CAM

Rôle : C'est le cerveau du système. Il gère la détection des obstacles via la caméra et prend des décisions en temps réel pour envoyer des alertes.

Fonction : Analyse des images capturées pour détecter des obstacles ou des trous, en utilisant des algorithmes d'IA ou de traitement d'image.

Carte mère

DFPlayer Mini

Rôle : Lecteur audio qui permet d'émettre des alertes vocales.

Fonction : Lorsqu'un obstacle est détecté, il joue des fichiers audio MP3 stockés sur la carte microSD pour avertir l'utilisateur (ex : "Obstacle détecté devant, contourner").

Actionneur

Haut-parleur

Rôle : Restituer le son des alertes émises par le DFPlayer Mini.

Fonction : Reproduit les messages vocaux ou les sons d'alerte qui indiquent à l'utilisateur la présence d'obstacles ou de dangers.

Actionneur

Carte microSD

Rôle : Stocke les fichiers audios nécessaires aux alertes vocales.

Fonction : Permet de charger plusieurs fichiers audios pour différents types d'alertes (obstacles, terrains accidentés, etc.).

Stockage

Batterie Li-ion (7,4V)

Rôle : Fournit de l'énergie à l'ensemble du système.

NB : La tension de la batterie n'est pas fixe et peut être changée avec le temps.

LE RÔLE DE L'IA AU CŒUR DU BINV

Intelligence Artificielle

L'IA est au cœur du système BINV. Elle permet de voir à la place de la personne non voyante, en utilisant la caméra pour reconnaître les obstacles ou les trous, puis donne un avertissement vocal pour l'aider à éviter le danger.

Fonctions de l'IA :

  • Analyser en temps réel les images capturées par la caméra de l'ESP32-CAM.
  • Reconnaître les éléments de l'environnement (obstacles, trous, sol libre).
  • Prendre une décision automatique : alerter l'utilisateur en cas de danger ou obstacle.
  • Générer une alerte adaptée (message vocal via DFPlayer Mini).

Exemple concret : L'IA voit une forme sombre devant → elle la reconnaît comme un trou → elle déclenche l'audio : "Attention, trou devant".

NB : L'IA est entraînée pour reconnaître différents types d'obstacles et adapter le message vocal en conséquence.

ÉVOLUTIONS ET AMÉLIORATIONS

Le projet BINV est conçu pour être évolutif. Voici les améliorations envisagées :

1

Version Bluetooth

Intégration d'un module Bluetooth pour connecter des écouteurs sans fil, offrant plus de liberté à l'utilisateur.

2

Reconnaissance avancée

Amélioration de l'IA pour reconnaître des objets spécifiques (escaliers, poteaux, piétons, etc.) et fournir des alertes plus précises.

3

GPS intégré

Ajout d'un module GPS pour aider l'utilisateur à se repérer et à naviguer dans des itinéraires connus.

4

Recharge solaire

Panneau solaire intégré pour une autonomie prolongée, idéal pour les zones sans électricité.
