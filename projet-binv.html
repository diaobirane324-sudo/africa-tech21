Bâton Intelligent pour Non-Voyants

BINV_V360 - Innovation Technologique au Service de l'Accessibilité

Projet révolutionnaire utilisant l'IA pour assister les personnes non-voyantes dans leurs déplacements

Description du Projet

Objectif : Faciliter la vie des personnes non-voyantes en développant un bâton intelligent capable de détecter les obstacles et de fournir des alertes vocales multilingues.

Fonctionnalités :

  • Détection d'obstacles (trous, murs, roches, etc.)
  • Alertes vocales en français, wolof, anglais
  • Écouteur filaire pour une interprétation précise
  • Intelligence artificielle pour l'analyse en temps réel

Composants Techniques

  • ESP32-CAM (Carte Mère)
    Cerveau du système - Analyse les images et prend des décisions en temps réel
  • DFPlayer Mini
    Lecteur audio pour les alertes vocales
  • Haut-parleur
    Restitution des alertes sonores
  • Carte microSD
    Stockage des fichiers audio multilingues
  • Batterie Li-ion 7,4V
    Alimentation de l'ensemble du système

Code Arduino - Rénovation Prototype

#define trigPin 9
#define echoPin 10
#define irPin 8
#define buzzer 7

long duration;
int distance;

void setup() {
pinMode(trigPin, OUTPUT);
pinMode(echoPin, INPUT);
pinMode(irPin, INPUT);
pinMode(buzzer, OUTPUT);
Serial.begin(9600);
}

void loop() {
// Mesure de distance avec ultrason
digitalWrite(trigPin, LOW);
delayMicroseconds(2);
digitalWrite(trigPin, HIGH);
delayMicroseconds(10);
digitalWrite(trigPin, LOW);
duration = pulseIn(echoPin, HIGH);
distance = duration * 0.034 / 2;

// Lecture capteur infrarouge
int irValue = digitalRead(irPin);

// Détection obstacle (moins de 50 cm ou obstacle infrarouge)
if (distance < 50 || irValue == LOW) {
digitalWrite(buzzer, HIGH); // Alerte
} else {
digitalWrite(buzzer, LOW); // Rien
}
delay(200);
}
